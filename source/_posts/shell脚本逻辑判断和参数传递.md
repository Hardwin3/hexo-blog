# shell脚本逻辑判断和参数传递

#### 向脚本传递参数

- 位置参数，ls /etc ,/etc就是命令传递进去的参数

- $1,$2,$3…${10},${11}

- shift n ，轮替前n个参数，踢掉

- 特殊变量
$0,脚本路径本身

- $#,传递参数个数

- $@，所有参数

- $*,所有参数





#### 执行顺序

- 选择执行



单分支



```shell
if 测试条件；then （then可以写在下面,不带;）
代码分支
fi
```



- 双分支



```sh
if 测试条件；then
分支1
else
分支2
fi
```
